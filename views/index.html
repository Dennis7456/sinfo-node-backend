sinfo-node-backend

A node js backend

This is a backend REST api that consumes data from JSONPlaceholder. This application is built with ExpressJS and and NodeJS and is available on Node-Backend-Link. The source code is available on Github

Links to fetch data:

Users

  • Get all users
  • Get a single user

Posts

  • Get all posts
  • Get a single post
  • Delete a single post e.g post 7
  • Get a single user's posts e.g user 7
  • Get comments related to a post e.g post 7

Comments

  • Get all comments
  • Get a single comment
  • Delete a single comment e.g comment 8
  • Get a single post's comments e.g post 8

Albums

  • Get all albums
  • Get a single album e.g album 12
  • Delete an album e.g album 9
  • Get a user's albums e.g user 9
  • Get an album's photos e.g album 17

Photos

  • Get all photos
  • Get a single photo e.g photo 5
  • Delete a single photo e.g photo 19
